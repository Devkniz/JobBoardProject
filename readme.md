## Getting Started

### Prerequisites

To run this project, you will need:

- [Node.js](https://nodejs.org/)
- [npm](https://www.npmjs.com/)

### Installation

1. Clone the repository:

  git@github.com:Devkniz/JobBoardProject.git

2. Run the project:

Linux: ./start.sh
Windows: ./start.bat
